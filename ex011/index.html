usando link externos

Você pode acessar um link aqui com a tag "a"

Você também pode acessar outro link aqui para abrir nova aba com o site, adicionar tag "target="_blank"" e "rel="external"" após o link

Usando links internos

para ir para outra página basta usar a tag "a" e colocar o nome do arquivo em href clique aqui

Você também pode acessar paginas de outro pastas dentro da pasta principal, usando CTRL+ESPAÇO nas "" de href
